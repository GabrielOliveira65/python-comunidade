Projeto Comunidade Postify

Este projeto foi desenvolvido durante meus estudos em desenvolvimento web, como parte de um curso prático focado na criação de aplicações completas.

A proposta foi construir uma plataforma de posts (estilo blog), utilizando o microframework Flask em conjunto com SQLAlchemy para gerenciamento do banco de dados.

Durante o desenvolvimento, foram aplicados conceitos essenciais como:

* Estruturação de rotas e templates
* Integração com banco de dados
* Sistema de autenticação e controle de acesso
* Manipulação de dados dinâmicos

Para a interface, foi utilizado Bootstrap, permitindo a criação de um layout responsivo e organizado.

Este projeto representa a base do meu aprendizado prático, e continuará sendo evoluído com novas funcionalidades e melhorias.
